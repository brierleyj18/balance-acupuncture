Wix Embed Guide

Balance Acupuncture — how to install each snippet into your Wix site

How this works

The snippets in this folder are self-contained HTML blocks, each designed for a specific section of your Wix site. There are two types:

  • Head code — added once to your whole site via Wix Dashboard → Settings → Custom Code. This sets up the fonts and colour palette globally.
  • HTML iframe embeds — added to individual pages via Wix Editor → Add → Embed → HTML iframe. Each is a complete section (conditions grid, testimonials, etc.).
Start with the head code first. Install wix-head-code.html before adding any section embeds, so the fonts load correctly across your site.

Step 1 — Install the head code

Wix Dashboard → Custom Code

This adds the Google Fonts (Cormorant Garamond + Inter) and the colour palette variables to every page of your site.

  1. Go to your Wix Dashboard (not the Editor — the dashboard at manage.wix.com)
  2. Click Settings in the left sidebar
  3. Scroll down to Custom Code and click it
  4. Click + Add Custom Code
  5. Open the file wix-head-code.html from this folder, copy all the code inside it
  6. Paste it into the code box in Wix
  7. Set Place Code in to Head
  8. Set Add Code to Pages to All Pages
  9. Give it a name like "Balance Design System" and click Apply
Open wix-head-code.html

Step 2 — Adding section embeds

Wix Editor → HTML iframe

Each section embed below follows the same installation process:

  1. Open the page in Wix Editor where you want the section to appear
  2. Click Add (+) in the left panel
  3. Go to Embed Code → Embed HTML (sometimes listed as "HTML iframe")
  4. Drag the element onto your page
  5. Click the element, then click Enter Code
  6. Open the relevant embed file, copy all the code, and paste it in
  7. Click Update to preview
  8. Resize the iframe height in Wix to match the content height shown in the table below
  9. Set the iframe width to 100% by stretching it edge to edge
Width tip. After placing the iframe, right-click it → "Stretch to full width" or drag the edges to the full page width. Each embed is designed to fill its container naturally.

Helpful tips

Mobile preview. After adding each embed, switch to mobile view in the Wix Editor to check it displays correctly. The embeds are all responsive, but you may need to adjust the iframe height separately for mobile.
Background colours. Each embed has a transparent or white background. If a section embed has a cream background (like the conditions grid), make sure the Wix section strip behind it matches — use hex #f9f6f1 in Wix's colour picker.
Fonts not loading? Make sure wix-head-code.html was installed in Custom Code → Head before previewing. If fonts still don't load inside the iframe, each embed also includes its own local font link as a fallback.
